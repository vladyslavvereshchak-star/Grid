# GRID — Desktop App (Windows)

## Структура файлов

```
grid/
├── main.js          ← Electron главный процесс (новый)
├── preload.js       ← Мост renderer↔main (новый)
├── server.js        ← Твой сервер (не трогаем)
├── package.json     ← Обновить (см. ниже)
├── public/
│   ├── index.html   ← Обновлённый (новый)
│   └── icon.png     ← Иконка приложения (нужно добавить)
```

## Шаги установки

### 1. Обнови package.json
Замени содержимое своего package.json на содержимое package-electron.json

### 2. Установи зависимости
```
npm install
```

### 3. Добавь иконку
Положи файл icon.png (256x256) в папку public/
Можно сделать на: https://favicon.io или любой онлайн конвертер

### 4. Запуск для теста
```
npm start
```
Откроется окно приложения. Сервер запускается автоматически внутри.

### 5. Сборка .exe установщика
```
npm run build
```
Готовый установщик появится в папке dist/

### Горячая клавиша
Alt+G — показать/скрыть окно GRID
