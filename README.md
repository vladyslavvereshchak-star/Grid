# Grid — Локальный сервер

## Что нужно
- Node.js (скачать на nodejs.org, версия 18+)

## Запуск

1. Распакуй архив в любую папку
2. Открой терминал (cmd / PowerShell) в этой папке
3. Установи зависимости (один раз):
   ```
   npm install
   ```
4. Запусти сервер:
   ```
   node server.js
   ```
5. Открой браузер: http://localhost:3000

## Как подключить друга по сети

1. Узнай свой локальный IP:
   - Windows: открой cmd → введи `ipconfig` → смотри IPv4 Address
   - Mac/Linux: `ifconfig` или `ip a`

2. Скинь другу адрес вида: `http://192.168.1.XX:3000`
   (он должен быть в той же Wi-Fi сети)

3. Друг открывает этот адрес в браузере — и вы в одном чате
